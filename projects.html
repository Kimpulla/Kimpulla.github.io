Here are some of the projects I have worked on:

Guilt Tripper

Guilt Tripper is a Runelite plugin designed for Old School Runescape, adding a unique twist to the gaming experience. This plugin periodically sends messages to players, humorously "accusing" them of playing too much and suggesting they take a break. And much more!

Github

PDF Roulette

PDF Roulette is a straightforward application designed to provide a random page from a PDF file, akin to drawing a random card in card games. Whether you're looking for a serendipitous read or just want to explore your PDFs in a new way, PDF Roulette offers a quick and easy solution.

Github

Cyber Security Base 2023 Project

Assignment from the course Cyber Security Base 2023 to create a web application that has at least five different flaws from the OWASP top ten list as well as their fixes. The application should have a backend.
